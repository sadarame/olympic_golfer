## Running the Application

To run the application on an iOS simulator:

1.  **Launch the iOS Simulator:**
    ```bash
    open -a Simulator
    ```
2.  **Run the Flutter Application:**
    Navigate to the `majong_paper` directory and run:
    ```bash
    flutter run
    ```
